ACF support for selected Gutenberg blocks. This will add an option in block editor toolbar to replace content with ACF field values.

== Description ==

ZIOR Block Elements is a WordPress plugin that gives editors an option to replace block content with ACF values in gutenberg editor toolbar.

Features

* Custom block elements
* ACF support

Read documentation <a href="https://ziorweb.dev/docs/zior-block-elements.html" target="_blank">here</a>

== Frequently Asked Questions ==

= Default settings =

When activated, a button will be added to the Gutenberg editor toolbar to be able to replace content with ACF field values.
